Plane Radar Page Button Wiring

ESP32-C3 Super Mini + momentary pushbutton for manual page cycling
What GPIO2 means on your board: look for the tiny printed label 2, IO2, or GPIO2. On the common ESP32-C3 Super Mini layout, it is on the same edge as 5V, GND, 3V3, 4, 3, 2, 1, 0. It sits between 3 and 1.

2. Breadboard hookup

Do not wire the button to 5V. The firmware uses the ESP32 internal pull-up, so the button only needs GPIO2 and GND.
Boot note: GPIO2 can be strap-sensitive on ESP32-C3 boards. Do not hold the button down while plugging in USB or pressing reset. If the board ever refuses to boot when the button is held, release it and reset again.

3. How to identify GPIO2

  1. Unplug USB power first.
  2. Find the ESP32-C3 Super Mini board, not the round display.
  3. Look for the edge that has labels like 5V, GND, 3V3, 4, 3, 2, 1, 0.
  4. The pad labelled 2 is GPIO2. It is normally between 3 and 1.
  5. If your clone says IO2 or GPIO2, that is the same thing.
