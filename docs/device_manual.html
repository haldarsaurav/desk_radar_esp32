2 · Wiring schematic

Display on hardware SPI. Button optional — or use the onboard BOOT button (GPIO9), no soldering.

GC9A01 pinESP32-C3 pin
VCC3V3 — never 5V!
GNDGND
SCL / SCKGPIO4
SDA / MOSIGPIO3
DCGPIO10
CSGPIO1
RST / RESGPIO0
Page buttonGPIO2 → GND (or GPIO9 = BOOT)
⚠ Display VCC must go to 3V3. 5 V destroys the GC9A01.
BOOT button as page button: works fine after boot (set PAGE_BUTTON_PIN 9), but holding it while plugging in USB starts the ESP32-C3 bootloader — release and re-plug.

3 · Controls & pages

ActionEffect
Short pressNext page, carousel pauses on it
Long press (1.2 s)Resume automatic page rotation

Page order: HOME RADAR → TRAFFIC BRIEF → NEAREST TRACK → COOLEST TRACK → MUC MAP → WEATHER → LEGEND. Rare aircraft or emergencies jump the display to the COOLEST page automatically.
